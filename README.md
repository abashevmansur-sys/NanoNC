# NanoNC
Minimalist Wi-Fi-based CNC controller for ESP32 with an intuitive web interface for remote control and G-code execution.
NanoNC — это ультра-минималистичная прошивка для ЧПУ-станков на базе контроллера ESP32. Проект создан для тех, кто ценит простоту, скорость и изящный пользовательский опыт без лишнего «раздутого» функционала.
🚀 Основная идея
Вместо использования тяжелых и сложных систем управления, NanoNC превращает ваш ESP32 в полноценный веб-контроллер станка, который работает прямо из браузера через Wi-Fi, а так же через отдельное Android-приложение.
✨ Ключевые особенности
Минимализм и эффективность: Прошивка использует асинхронный веб-сервер и аппаратные таймеры ESP32 для точного управления шаговыми двигателями, обеспечивая плавную и стабильную работу.
Изящный Web-интерфейс:
Адаптивный дизайн, идеально подходящий для управления станком прямо со смартфона.
Интуитивное управление джойстиком для перемещения осей X, Y и Z.
Визуализация процесса выполнения G-Code файлов непосредственно с SD-карты.
Реализована поддержка темной темы, соответствующей современным стандартам.
Простота запуска:
Станок создает собственную точку доступа (Wi-Fi), что позволяет управлять им без внешней сети.
Минимальное количество сторонних зависимостей.
Функциональный минимум:
Поддержка G-Code (G0, G1, G92).
Работа с файлами через SD-карту.
Реальное время: мониторинг координат и статуса исполнения через WebSocket.
🛠 Технический стек
Hardware: ESP32, драйверы шаговых двигателей, SD-карта.
Firmware: C++ (Arduino IDE), ESPAsyncWebServer, AsyncTCP.
Frontend-1: HTML5, CSS (Flex/Grid, dvh), Vanilla JavaScript, WebSockets.
Frontend-2: Flutter, WebSockets.
📋 Как использовать
Загрузите прошивку в ESP32 через Arduino IDE.
Подключитесь к Wi-Fi сети с именем NanoNC (пароль 12345678).
Откройте браузер и перейдите по адресу http://192.168.4.1/.
Загрузите ваши файлы G-Code на SD-карту и приступайте к работе!
